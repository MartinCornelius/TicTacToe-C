# **TicTacToe-C**
Tic tac toe console game written in c with a simple **minimax algorithm** for ai. 

### **How to play**
- Run game.exe
- Game launches in console
- Use numbers 1 - 9 to play
- Close the console window or press Ctrl + c to break it
